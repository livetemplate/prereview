# Quiz me on this diff

Generate a comprehension quiz about this file's diff so I can check whether I
actually understood the change before accepting it.

Do **not** edit files and do **not** propose suggestions — answer by running
`prereview quiz` (not `prereview suggest`), then mark this comment done and reply
with a one-line summary.

3–5 multiple-choice questions, grounded strictly in this diff, each anchored to
the lines it asks about (`"kind": "line"`), or to the change as a whole
(`"kind": "file"`) when it is about something absent. Include at least one
`decision` question — something you chose that I never asked for — if this diff
contains one. The full schema and the probe kinds are in the prereview skill.
